Overview
LibraryProject is a Django web application project created as part of the Django learning curriculum. This project serves as a foundation for building Django-based web applications with a library management focus.

Setup Instructions
Prerequisites
Python 3.x installed on your system
pip (Python package manager)

Installation Steps,
Install Django,
Create the Django Project,
Navigate to Project Directory,
Run the Development Server, and finally
Verify Installation

Development Server
The development server runs on http://127.0.0.1:8000/ by default. It automatically reloads when you make changes to your code, making it ideal for development.
